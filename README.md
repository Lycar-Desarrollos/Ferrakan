# Ferretería Ferrakan - Sitio en Construcción

Este repositorio contiene el código fuente para la página de inicio (landing page) temporal de la ferretería **Ferrakan**.

## Características del Sitio
- **Cuenta Regresiva**: Un temporizador dinámico en JavaScript que muestra el tiempo restante para la inauguración oficial.
- **Formulario de Suscripción**: Los visitantes pueden ingresar su dirección de correo electrónico para recibir notificaciones y ofertas exclusivas de lanzamiento.
- **Detalle de Servicios**: Tarjetas interactivas con efectos de estilo modernos que describen las principales ofertas de Ferrakan (herramientas, materiales de construcción, pinturas, y garantía).
- **Diseño Premium**: Interfaz oscura moderna (*dark mode*), luces de acento ambiental animadas, cuadrícula de fondo, y un diseño totalmente responsivo adaptado para dispositivos móviles, tabletas y computadoras de escritorio.

## Tecnologías Utilizadas
- HTML5 Semántico
- CSS3 (Variables personalizadas, Flexbox, CSS Grid, Glassmorphic Glass, keyframes de animación)
- JavaScript Vanilla
- [Lucide Icons](https://lucide.dev/) (Cargado mediante CDN)
- Google Fonts (Fuentes *Outfit* y *Plus Jakarta Sans*)

## Estructura del Proyecto
- `index.html` - Código principal y lógica JavaScript.
- `styles.css` - Estilos del sitio y animaciones.
- `.gitignore` - Archivos excluidos del control de versiones.
